# Lunar Folk Festivals & Anniversaries

## Retrieve Lunar Festivals Within a Date Range

```js
import { getLunarFestivals } from "vietnamese-days";

/**
 * Retrieve lunar festivals (includes fixed festivals and specially calculated festivals)
 * @param [start] Start date (default: today)
 * @param [end] End date (when omitted, only queries the start date)
 */
getLunarFestivals('2026-02-17'); // Lunar New Year
getLunarFestivals('2026-02-17', '2026-02-20');
```

### Common Lunar Folk Festivals & Anniversaries

#### 1st Month
- **1st Day**: Lunar New Year (Tết Nguyên đán)
- **15th Day**: First Full Moon Festival (Tết Nguyên tiêu)

#### 3rd Month
- **3rd Day**: Cold Food Festival (Tết Hàn thực)
- **10th Day**: Hung Kings Commemoration (Giỗ Tổ Hùng Vương)

#### 4th Month
- **8th Day**: Buddha's Birthday (Vesak - Mahayana tradition)
- **15th Day**: Buddha's Birthday (Vesak - Official/Theravada tradition)

#### 5th Month
- **5th Day**: Double Fifth Festival (Tết Đoan ngọ)

#### 7th Month
- **15th Day**: Ghost Festival / Ullambana (Rằm tháng Bảy / Lễ Vu Lan)

#### 8th Month
- **15th Day**: Mid-Autumn Festival (Tết Trung thu)

#### 9th Month
- **9th Day**: Double Ninth Festival (Tết Trùng cửu)

#### 10th Month
- **10th Day**: New Rice Festival (Tết Thường tân)
- **15th Day**: Lower Yuan Festival (Lễ hội Hạ nguyên)

#### 12th Month
- **23rd Day**: Kitchen Guardians Day (Tết Táo quân)
- **Last Day**: Lunar New Year's Eve (Trừ tịch)
